Leyes de gestalt:
1- relación figura-fondo
2- igualdad o equivalencia -> ocurren varios elementos = agrupamos los similares para separarlos
3- Cierre  -> una figura cerrada es más fácil de entender, si no está cererada, solemos cerrarla mentalmente
4- Experiencia -> nuestra xp condiciona nuestra percepción
5- Semejanza -> elementos aislados pero similares son percibidos como parte del mismo grupo
6- Proximidad -> elementos aislados pero con cercanía son considerados como grupos
7- Simetría -> als imágenes simétricas son percibidas como iguales por 1 elemento de por medio / MAS RELACIONADAS ENTRE SÍ
8- Continuidad -> los elementos colocados forman un flujo orientado hacia algún lado, se perciben como un todo
9- Dirección común -> varios elementos aislados con movimiento común son considerados como grupos
10- Buena forma o Pregnancia -> simnplificamos lo que percibimos
11- Simplicidad -> se percibe todo de la forma más simple, aún cuando la forma es compleja o IMPOSIBLE nuestro breain la construye rápidamente
